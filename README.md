The goal of bbvmisc is to provide useful miscellaneous functions for the
BBV team at Public Health Scotland and Glasgow Caledonian University to
use.

## Installation

You can install the development version of bbvmisc from
[GitHub](https://github.com/) with:

``` r
# install.packages("pak")
pak::pak("alan-y/bbvmisc")
```

Note that if bbvmisc is stored on a private Github repository, you will
need to set up a [Github Personal Access Token
(PAT)](https://docs.github.com/en/authentication/keeping-your-account-and-data-secure/managing-your-personal-access-tokens)
before you can install the package. The PAT can be entered into the
`remotes::install_github()` function using the `auth_token` option or
[set up in other
ways](https://usethis.r-lib.org/articles/git-credentials.html) before
trying to install the package.
